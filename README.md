Yoga: All For Us is an interactive web application built to make daily yoga practice easy to follow, analyze, and track. It features a curated library of poses with clear guides, alongside an AI practice module that uses your webcam to monitor joint angles and score posture accuracy in real-time. A clean dashboard displays your progress, streaks, and visual consistency using an interactive 180-day calendar consistency heatmap. Developed using Node.js and MongoDB, the app also includes a resilient local database fallback so it stays fully functional offline.
